# 仪表盘介绍：

推荐使用三种仪表盘工具，分别是对应是 Kubernetes Dashboard 、 kube-explorer 和 Rancher UI ，其各自各有优劣。

  1. Kubernetes Dashboard
  2. kube-explorer
  3. Rancher UI
